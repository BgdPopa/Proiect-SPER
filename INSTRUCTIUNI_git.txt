Push către GitHub (repo gol sau deja inițializat)
===============================================

1) Deschide PowerShell în folderul proiectului:
   cd "C:\Users\bogdi\OneDrive\Desktop\ProiectSPER"

2) Inițializare git (dacă nu există deja .git în acest folder):
   git init
   git branch -M main

3) Legătură la remote (înlocuiește URL-ul dacă e alt repo):
   git remote add origin https://github.com/BgdPopa/Proiect-SPER.git

   Dacă primești eroare „remote origin already exists”:
   git remote set-url origin https://github.com/BgdPopa/Proiect-SPER.git

4) Primul commit și push:
   git add .
   git commit -m "SPER: Dubins + TSP nearest neighbor, notebook și figuri"
   git push -u origin main

5) Dacă GitHub are deja un commit (ex. README creat pe site) și push-ul e respins:
   git pull origin main --rebase
   git push -u origin main

Autentificare: pe GitHub se folosește de obicei un Personal Access Token în loc de parolă la HTTPS,
sau SSH (cheie) dacă ai configurat remote SSH.
